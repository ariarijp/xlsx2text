# xlsx2text

## Installation

```shell
$ go get github.com/ariarijp/xlsx2text
```

## Usage

```shell
$ xlsx2text -h
Usage:
  xlsx2text <FILE> <SHEET> [flags]

Flags:
  -h, --help         help for xlsx2text
  -s, --sep string   separator (default "\t")

$ xlsx2text Financial\ Sample.xlsx Sheet1
```
